# DedRoot Neovim IDE

Personal Neovim configuration for C, Rust, Linux, UNIX and terminal workflow.

## Installation

Clone repository:

```bash
git clone https://github.com/Punk-Zuckerberg/dedroot-nvim.git
cd dedroot-nvim
```

## macOS

```bash
make deps-macos
make install
```

## Arch Linux

```bash
make deps-arch
make install
```

## Manjaro

```bash
make deps-manjaro
make install
```

## Ubuntu

```bash
make deps-ubuntu
make install
```

## Debian

```bash
make deps-debian
make install
```

## Fedora

```bash
make deps-fedora
make install
```

## openSUSE

```bash
make deps-opensuse
make install
```

## Update plugins

```bash
make sync
```

## Clean Neovim cache

```bash
make clean
```
